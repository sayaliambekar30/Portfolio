Team Size: 4
Duration: 9 Months

"The Roomie" is a machine learning-based web application designed to help users find the perfect roommate match. Leveraging advanced algorithms, the application ensures compatibility and convenience for users seeking shared accommodation.

Key Features:
User Registration and Profile Creation: Secure registration process with detailed profile creation to capture user preferences and lifestyle habits.
